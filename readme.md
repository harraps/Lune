# Lune

A basic network project made with LÖVE and MoonScript.

## Compile

run the command
```
moonc -t build src/.
```
in the *root* directory of the project.

## Execute

run the command
```
love build server
```
to launch the server.

Then run the command
```
love build
```
to launch a client.
